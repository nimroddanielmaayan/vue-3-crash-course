# vue-crash-course

A Vue.js practice project

Course link - https://www.youtube.com/watch?v=KTFH4P8unUQ

## Customize configuration

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
